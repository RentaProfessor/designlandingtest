# Hideaway Design Collective Website

A beautiful, responsive landscape design company website featuring modern design and smooth user experience.

## Features

- **Responsive Design**: Works seamlessly on all devices
- **Portfolio Gallery**: Interactive image gallery with smooth zoom functionality
- **Contact Integration**: Direct contact information and form
- **Modern Styling**: Clean, professional design with smooth animations
- **Optimized Performance**: Fast loading and efficient code

## Files Included

- `index.html` - Main website file with all HTML, CSS, and JavaScript
- `*.PNG` - Portfolio images for the gallery
- `HIDEAWAY (1)_edited.avif` - Company logo
- `.gitignore` - Git ignore file for proper version control

## Local Development

To run this website locally:

1. Clone the repository
2. Navigate to the project directory
3. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```
4. Open your browser to `http://localhost:8000`
